# Debuilder
Боб строитель окончательно ёбнулся и жидко навайбкодил скрипт для создания стендов с документацией.
SVG формат работает на религиозных убежедниях, по-этому достаточно хорошо импортируется в братский CorelDraw, но это не точно у афтора версия 2025.
Не пытайтесь предъявлять претензии что это не CAD и вообще можно было бы интерфейс про приятнее, их никто не услышит. Юзабилити был пожертвован в фонд создания анальных пробок с трекболлом.
